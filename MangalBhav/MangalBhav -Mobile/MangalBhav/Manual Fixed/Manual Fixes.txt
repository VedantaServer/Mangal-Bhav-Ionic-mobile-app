Manual fixes :

1- Added this fix in to handle https call in  network_security_config.xml

add this key  in application tag in android:networkSecurityConfig="@xml/network_security_config" 

<application android:hardwareAccelerated="true" android:icon="@mipmap/icon" 
android:label="@string/app_name"
 android:networkSecurityConfig="@xml/network_security_config" 
 android:supportsRtl="true">

Create this file as network_security_config.xml under ...\platforms\android\app\src\main\res\xml

<network-security-config>
  <!--Set application-wide security config using base-config tag.-->
  <base-config cleartextTrafficPermitted="true"/>
</network-security-config>

2. Location Permission
Add these line in 

<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />

 
3. Copy root color file in android path (Merge platforms\android\app\src\main\res\values\color.xml) of barcodescanner & firebase.

4.AFter installing the Android.. run following plugin

ionic cordova plugin add cordova-plugin-firebase
cordova plugin add cordova-android-firebase-gradle-release


ionic cordova platform remove android
ionic cordova platform add android
